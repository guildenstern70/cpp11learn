cpp11learn
==========

A collection of simple source code examples in C++ 11, featuring key aspects of the language.

Build using CMake
-----------------

1. Make the CMake environment files:

       cd [cpp11learn_dir]  
       cmake .  

2. Make the project 

       cmake --build .
